# Nasdaq-Bloomberg Analysis

## Overview
This repository contains scripts and tools for analyzing stock market data from Nasdaq and Bloomberg. It includes:

- Data collection and processing scripts for Nasdaq and Bloomberg financial datasets.
- Dividend and stock projection models.
- Quantum finance simulations (e.g., regime analysis, portfolio optimization).
- Visualizations for market trends and strategy backtesting.
- Streamlit dashboards for interactive analysis.

## Features
- Import and process historical stock and dividend data.
- Predict stock trends and dividend projections.
- Quantum-based regime and portfolio analysis.
- Ready-to-use dashboards for visualization.
- Integration with multiple financial datasets.

## File Structure
- `*.py` → Python scripts for data analysis and modeling.
- `*.csv` → Historical stock and dividend datasets.
- `*.png` → Visualizations and charts.
- `quantum-bloomberg-db2/` → Quantum finance simulations (submodule/repository).
- `streamlit_*` → Streamlit dashboards for interactive analysis.
